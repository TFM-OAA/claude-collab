---
description: Снять свой LOCK-замок проекта (WORKING_NOW.txt)
---
Сними мой LOCK-замок проекта. Запусти PowerShell-скрипт и покажи его вывод:

```
powershell.exe -NoProfile -ExecutionPolicy Bypass -File ".\.claude\hooks\remove-lock.ps1"
```

Скрипт удалит `WORKING_NOW.txt`, только если замок принадлежит текущему пользователю (или владелец не указан). Чужой замок он не трогает. Скрипт сам определяет папку проекта, env-переменная не нужна. Кратко сообщи результат пользователю по-русски.
